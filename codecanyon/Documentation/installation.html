Installation

Build in auto installation [Recommended]
  1. Unzip the download file from codecanyon. You will find three folders, application, documentation and sql.
  2. Go to application file and zip all files under application folder. (if you are using unix like operating system, press Ctrl+H/ Command +H to show dotted files)
  3. Upload the zip file to your c-panel public_html and extract it
  4. Open a new tab and go to your web domain. xPOS automatically redirect you to the installation process
  5. Follow the steps and setup xPOS
Manual installation
  1. Unzip the download file from codecanyon. You will find three folders, application, documentation and sql.
  2. Go to application file and zip all files under application folder.(if you use unix like operating system, press Ctrl+H/ Command +H to show dotted files)
  3. Upload the zip file to your website public_html and extract it
  4. Edit .env as your need (Avoid any kind of spach during edit)
  5. in .env Make sure the variables DATABASE, MAIL,INSTALL,ADMIN is equals to 1
  6. Import the xpos.sql file form Sql folder that i include
