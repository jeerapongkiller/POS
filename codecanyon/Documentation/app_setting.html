App Setting

You will find app setting during xPOS installation process. you can able to change setting after install. Both are almost same process

Database Setup
Note :
You can access database setup during installation period only
  1. MySQL database setup (Only accessible during installation period)
  2. SMTP mail setup (You can skip it or edit it any other time)
  3. Admin Account setting (Only accessible during installation period a)
  4. Localization setting i.e : Timezone, Currency Code and Symbol
  5. Tax Setup
  6. Finish setup
  7. Enter you mysql host ip address (Default : 127.0.0.1)
  8. Enter your mysql port address (Default : 3306)
  9. Enter your mysql database name that you created for install xpos
  10. Enter your mysql user name
  11. Enter your mysql user password
  12. Enter your mysql user password again
  13. Click to submit so it will create requires tables automatically to your datbase

Note: After sucessfuly setup your database you will see smtp mail setting page. or it will return an error. If your get any error please try to setup databse agian

Mail Setup
  1. Enter your smtp mail host
  2. Enter your mail port
  3. Enter your email account
  4. Enter your email password
  5. Enter your email encryption type
  6. Click to save mail setting
  7. Click to skip this proces if you don't want to setup your mail
  8. Admin Account setting (Only accessible during installation period a)
  9. Localization setting i.e : Timezone, Currency Code and Symbol
  10. Tax Setup
Admin Account Setup
  1. Select your profile picture (Not Mandatory)
  2. Enter your name
  3. Enter your email address that will need for login
  4. Enter your login password
  5. Enter your login passowrd again
  6. Click to submit to create an admin.
  7. Localization setting i.e : Timezone, Currency Code and Symbol
  8. Tax Setup
Localization Setup
  1. Click to find your timezone formate
  2. Enter your timezone with proper php timezone format
  3. Enter your currency symbol
  4. Click to save
  5. Tax Setup
Tax Setup
  1. Enter your TIN number
  2. Enter Tax %
  3. Click to save
  4. SMTP mail setup (You can skip it or edit it any other time)
  5. Admin Account setting (Only accessible during installation period a)
  6. Localization setting i.e : Timezone, Currency Code and Symbol
