TAX

Tax Setup

login to your admin account and click on Setting from the left sidebar then click on App Setting then click on Tax Setup

  1. Enter your TIN number
  2. Enter Tax %
  3. Click to save
  4. SMTP mail setup (You can skip it or edit it any other time)
  5. Admin Account setting (Only accessible during installation period a)
  6. Localization setting i.e : Timezone, Currency Code and Symbol
