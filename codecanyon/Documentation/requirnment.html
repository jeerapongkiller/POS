Requirnment

Before start installation make sure you have full fill all the requirnment listed bellow

  • PHP 7.0 or later
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension
  • MYSQL DBMS
  • SMTP Mail server

Did you full fill theses requirnment ? Now your are free to go for installation.
