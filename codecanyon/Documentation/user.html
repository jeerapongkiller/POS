User Management

There are four kind of user in xPOS :

  1. Admin
  2. Outlet Owner / Manager
  3. Modarator &
  4. Sells Man

Outlet Owner / Manager and Modarator

Create Outlet Owner / Manager or Modarator
NOTE :

Only admin can create an outlet owner and modarator.

Only Outlet owner can able to create a sells man

    Login to your admin account

  1. Click on users button from the left sidebar
  2. Click on New user button from the left sidebar
  3. Click to add the new user's image
  4. Enter the name of new user
  5. Enter e-mail address of new user
  6. Enter the password for new user
  7. Enter the same password to confirm
  8. Select the type of new user
  9. Enter the phone number of new user
  10. Enter the additional phone number
  11. Enter the address of new user
  12. Click on save button to save the new user
Update Outlet Owner / Manager or Modrator
NOTE :

Only admin can create an outlet owner and modarator.

Only Outlet owner can able to create a sells man

  1. Click on Users button from the left sidebar
  2. Click on Users button from the left side bar
  3. Click on edit button to edit anything
Delete an Outlet Owner / Manager or Modarator
NOTE :

Only admin can delete an outlet
Once you delete an outlet you cannot able to recover it again

  1. Click on Users button from the left sidebar
  2. Click on Users button from the left side bar
  3. Click on delete button to delete anything

Sells man

Create a sells man
NOTE :

Only admin can delete an outlet
Once you delete an outlet you cannot able to recover it again

  1. Click on 'Sells Man' button from the left sidebar
  2. Click on 'New sells Man' button from the left sidebar
  3. Click to add the new sells man image
  4. Enter the name of new new sells man
  5. Enter the e-mail address of new sells man
  6. Enter a password
  7. Enter the password again to confirm
  8. Select the outlet of new sells man
  9. Enter the phone number of new sells man
  10. Enter the additional phone number of new sells man
  11. Enter the address of new sells man
  12. Click on save button
Update sells man
NOTE :

Only admin can delete an outlet
Once you delete an outlet you cannot able to recover it again

  1. Click on 'Usres' button from the left sidebar
  2. Click on 'Users' option from the left sidebar
  3. Click on delet button to delete user's information

  1. Click on 'SellsMan' button from the left sidebar
  2. Click on 'Sells Man' option from the left sidebar
  3. Click on delete button to delete the data of the sellsman
